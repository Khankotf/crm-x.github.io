# CRM-X

CRM-X — прототип CRM для малого B2C-бизнеса. В системе уже есть лиды, сделки, контакты, компании, задачи, продукты, партии товаров, канбан, дашборды, отчеты, роли пользователей и планы продаж.

## Что есть сейчас

Это пока frontend-прототип:

- `index.html` — структура страниц.
- `styles.css` — внешний вид и адаптивность.
- `app.js` — бизнес-логика CRM, демо-данные, роли, отчеты, фильтры, карточки, таблицы и канбан.
- `src/storage.js` — единый слой хранения данных.
- `supabase/migrations/` — черновик миграций будущей базы данных.
- данные хранятся в браузере через `localStorage`.

Такой вариант подходит для демонстрации и проверки идеи, но для продажи нужно перейти к серверу, базе данных, настоящей авторизации, разделению данных клиентов и резервным копиям.

## Как запустить локально

Можно открыть `index.html` прямо в браузере. Лучше запустить простой локальный сервер:

```powershell
cd C:\Users\khank\VSCODE_Projects\Preject_X
python -m http.server 8080
```

Потом открыть:

```text
http://localhost:8080
```

Демо-логины:

- `vip / 123456`
- `leader / 123456`
- `alina / 123456`

## Проверка

Проверить JavaScript на синтаксические ошибки:

```powershell
node --check app.js
```
